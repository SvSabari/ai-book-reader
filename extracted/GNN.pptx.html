PROBLEM STATEMENT

Cyber attacks are increasing in smart industrial (IIoT) systems.
Traditional security systems cannot detect new or unknown attacks.
Existing systems often give many false alarms.
If one device is attacked, the threat can quickly spread to other connected devices.
Therefore, industries need a smart and predictive security system to detect threats early.

Slide 3

PROPOSED SOLUTION

Model the IIoT network as a graph where devices are nodes and their communications are edges.
Use a Graph Neural Network (GNN) to analyze relationships between connected devices.
Detect and classify cyber threats in real time based on abnormal communication patterns.

Slide 4

WHAT IS GRAPH NEURAL NETWORK ?

A Graph Neural Network (GNN) is an AI model that analyzes devices and their connections in a network.
WHAT ID DOES
It learns what "normal behavior“ of device looks like.
If something unusual happens → it raises an alert before it attack the connections of that Device.
